# devops-capstone-project

![Build Status](https://github.com/DMG202022/devops-capstone-project/actions/workflows/ci-build.yaml/badge.svg)

This project is part of a DevOps capstone lab. It focuses on building and deploying a customer accounts microservice using Agile practices, Docker, and Kubernetes.

The project includes:
- Agile planning with GitHub Projects
- User story creation and backlog management
- Microservice development
- Containerization using Docker
- Deployment to Kubernetes
